A.5 Metodologie di Calcolo

  • Scope 1 Combustione: fattori DEFRA (Gas Naturale, Gasolio, Benzina), AR6 GWP. Output CO2, CH4, N2O.
  • Scope 1 Processo: fattore stechiometrico 0,4397 tCO2/t CaCO3 (IPCC AR6); solo IANO.
  • Scope 1 Fugitive HFC: zero dichiarativo — impianti ad anello chiuso; top-up refrigerante dichiarati trascurabili (2026-05-13).
  • Scope 2 LB: fattore grid ISPRA Italia (primario), IEA (secondario).
  • Scope 2 MB: 0 tCO2e/MWh per volumi EE_Acquistata_GO; residual mix ISPRA per EE_Acquistata_Grid (SASSUOLO 2024).
  • Scope 3 Cat 1: mass-based (ecoinvent v3.10) + spend-based (EXIOBASE).
  • Scope 3 Cat 3: WTT DEFRA; quantità derivata da Σ Scope 1, non da CSV Cat 3.
  • Scope 3 Cat 4/9: distance-based (tkm × DEFRA freight factors per mode).
  • Scope 3 Cat 7: distance-based; FTE HR ufficiale 2024=506, 2025=484 (confermato 2026-05-13).

A.6 Anno Base e Politica di Ricalcolo

Anno base: 2024. Un cambiamento strutturale (acquisizione/dismissione) o una revisione metodologica che causa uno shift >5% nel totale Scope 1+2 tCO2e dell'anno base attiva il workflow obbligatorio di ricalcolo (FR-26).
